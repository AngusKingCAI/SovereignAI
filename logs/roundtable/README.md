# Round Table Score Documents

This directory contains persistent Round Table score documents for long-term tracking of panelist performance.

## Contents

- `roundtable-scores-plan-{N}.md` - Single-plan review scores
- `roundtable-scores-batch{N}-{M}.md` - Batch review scores (multiple plans reviewed together)

## Purpose

These documents enable long-term tracking of which panelists consistently provide the highest-quality findings. Use historical scores to optimize panel composition over time.

## Format

Score documents are created after the final Round Table round when a clean pass is achieved. They contain:

- Per-review verdicts
- Per-plan verdicts  
- Cross-reviewer comparisons
- Summary statistics
- Panelist performance notes

See `.agent/architect/AI_HANDOFF.md` Section 6 Step 6 for detailed format specifications.
